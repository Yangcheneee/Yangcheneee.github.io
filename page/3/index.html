博客（一）：从零开始搭建你的个人博客
发表于2023-08-21|博客
此文章将会指引你搭建hexo框架的博客，并将其部署到Git Hub上。 一. 环境搭建1. 安装Nodejs（npm）下载链接：Nodejs官方网站 这里直接选择官方推荐的稳定版本：Recommend For Most Users。 安装过程一路回车就好，默认选项不需要任何额外的操作。 安装完成以后，打开cmd。输入命令：node -v，若返回nodejs版本信息则安装成功。 nodejs内置npm。输入命令npm -v，返回npm版本信息则安装成功。 2. 安装GitGit下载网址 下载标注的版本。 安装一路回车就好。 二. 本地部署你的博客1. 安装Hexo，新建hexo框架项目打开cmd输入命令：npm install -g hexo-cli，等待安装完成即可。 安装完成以后输入命令：hexo -v查看版本，返回结果则安装成功。 选择一个用于放置博客文件的文件夹，在此目录下使用命令：hexo init myblog，新建你的博客项目。你可以看到在你的文件夹下多了一个myblog的项目。 输入命令：cd myblog进入myblog项目，安装npm：npm install即可 ...

如果你有任何问题，在评论提出 1. 博客？一些好看的博客：这里 我的个人博客：良月的小窝 你可以通过这个系列文章做出与我相似的个人博客，当然或许更好。 2. 个人博客有啥用？ 个人博客主要用于记录你的学习笔记，简单来说博客是一个在线记事本。 除此之外，个人博客可以用于知识分享，其他人可以通过网络访问你的博客。 抑或是用于满足好奇心，抑或是喜欢漂亮的博客页面…… 总之，搭建你的个人博客并不需要太多理由，只是去做也是一种乐趣！ 3. 搭建个人博客难吗？一点都不难！ 你可以不写任何代码，只是安装一些文件并配置它们就可以搭建属于你的个人博客。 4. 如何搭建我的个人博客？我采用GitHub Pages搭建hexo框架的butterfly主题博客，这是一种简单的方式。 大概流程：下载框架和主题文件；修改配置以美化你的博客界面；生成网页静态文件上传到Git Hub仓库使得其他人可以访问。 如果你感兴趣，我很乐意继续讲下去。
